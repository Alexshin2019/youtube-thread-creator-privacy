1. 수집하는 정보

YouTube Thread Creator는 다음과 같은 정보만을 수집합니다:

  • YouTube 영상 URL
  • 영상 제목

  • 조회수 정보

모든 정보는 공개된 YouTube 데이터에서만 수집되며, 사용자의 개인정보는 수집하지 않습니다.

3. 정보 보관 및 삭제

수집된 정보는 스레드 생성 즉시 처리되며, 별도로 저장되지 않습니다. 모든 처리는 실시간으로 이루어지며, 처리 완료 후 데이터는 즉시 삭제됩니다.

4. 제3자 제공

수집된 정보는 스레드 생성을 위한 AI 처리 목적으로만 사용되며, 다른 목적으로 제3자에게 제공되지 않습니다.
